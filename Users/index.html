№1: Функция sum
Недостаток:
Функция выполняет два действия: суммирование элементов массива и деление суммы на длину массива.
Имя функции не отражает точно то, что она делает.
Код функции слишком длинный.
Исправление:
function getSum(arr) {
return arr.reduce((a, b) => a + b);
}
В этом варианте мы используем метод reduce массива для вычисления суммы элементов. Это делает код более коротким и эффективным.
№2: Функция func
Недостаток:
Функция выполняет два действия: одна функция для каждого массива.
Имя функции не отражает точно то, что она делает.
Код функции слишком длинный.
Исправление:
function getSum(arr1) {
return arr1.reduce((a, b) => a + b);
}
function getSum(arr2) {
return arr2.reduce((a, b) => a + b);
}
В этом варианте мы вынесли каждое действие в отдельную функцию. Это делает код более коротким и эффективным.
