Пусть главная ось горизонтальна. Куда будет направлена поперечная ось?

Ответ: Поперечная ось будет направлена сверху вниз.

2: Пусть главная ось вертикальна. Куда будет направлена поперечная ось?

Ответ: Поперечная ось будет направлена слева направо.

3: Пусть поперечная ось направлена направо. Куда в таком случае может быть направлена главная ось?

Ответ: Главная ось может быть направлена сверху вниз (column) или снизу вверх (column-reverse).

4: Пусть поперечная ось направлена вниз. Куда в таком случае может быть направлена главная ось?

Ответ: Главная ось может быть направлена слева направо (row) или справа налево (row-reverse).

5: Может ли поперечная ось быть направлена справа налево?

Ответ: Нет, поперечная ось не может быть направлена справа налево. Она может быть направлена только сверху вниз или слева направо.

6: Может ли поперечная ось быть направлена снизу вверх?

Ответ: Нет, поперечная ось не может быть направлена снизу вверх. Она может быть направлена только сверху вниз или слева направо.

7: Пусть свойство flex-direction имеет значение row. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Ответ: Главная ось будет направлена слева направо, а поперечная ось — сверху вниз.

8: Пусть свойство flex-direction имеет значение column. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Ответ: Главная ось будет направлена сверху вниз, а поперечная ось — слева направо.

9: Пусть свойство flex-direction имеет значение row-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Ответ: Главная ось будет направлена справа налево, а поперечная ось — сверху вниз.

10: Пусть свойство flex-direction имеет значение column-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Ответ: Главная ось будет направлена снизу вверх, а поперечная ось — слева направо.
